# C：把代价算清楚

## 医美消费不是一笔账，是四笔账

价格最容易被看见，所以很多人以为预算够了，决定就可以做。

其实不够。一个决定至少要算四笔账：身体账、金钱账、时间账、退出账。

少算一笔，后面就可能用更多的钱和情绪补回来。

## 1. 身体账：我愿意承担什么不确定性

所有身体介入都不是把钱投进机器，等一个标准化结果。个体差异、恢复过程、效果边界、并发症处理，都属于决定的一部分。

面诊时要问的，不是「安不安全」这种只能得到安慰的话，而是更具体的问题：

- 这项治疗最常见的不舒服或恢复问题是什么？
- 哪些结果属于正常波动，哪些情况需要及时处理？
- 如果效果不符合预期，通常会怎么评估和处理？
- 我的工作、皮肤状态、既往治疗和生活安排，会不会改变这个判断？

听完后再问自己：这些答案我听懂了吗？没有听懂，就先别把「医生说没问题」当成知情。

## 2. 金钱账：别把分期当成变便宜

分期能降低一次付款时的痛感，却不会降低总成本。更麻烦的是，它会把「我现在愿不愿意承担这笔钱」变成「我每个月能不能挤出这一点」。

两句话差得很远。

做决定前，把项目费用之外的可能支出写出来：复诊、护理、请假、交通、后续调整，以及如果计划改变时已经付出去的钱。不是为了把人吓退，而是避免预算只够买开始，不够承担过程。

一个简陋但有效的办法：把总成本写成现金，而不是月供。再看一眼，心里有没有变化。

## 3. 时间账：恢复期不只是几天不能见人

恢复时间会碰到真实生活：会议、拍照、出差、孩子、相亲、节日、情绪承受力。

有的人技术上适合，时间上却不适合。比如刚好要面对重要工作节点，却没有空间处理肿胀、疼痛、反复担心或者别人追问。强行把恢复期塞进生活缝隙，焦虑会放大对效果的评价。

所以，别只问「多久恢复」。还要问：我有多少时间可以不安排重要社交；谁能在我不舒服时提供支持；如果恢复比预想慢，我的生活会不会被打乱。

## 4. 退出账：如果我后悔，路在哪里

人做决定时很少主动想后悔，销售也很少主动讲后悔。但退出方式决定了一个消费到底有多重。

退出账包括：是否可逆、调整难度、后续成本、发生争议时能保存哪些材料、出现不适时联系谁、能否获得第二意见。

不是所有项目都需要最坏打算。只是高代价的决定，不能没有退路。

如果咨询里只谈如何开始，从不谈如果不满意怎么办，这本身就值得记一笔。

## 一张很老派的表格

付款前，用纸把四笔账各写一行。写不满也没关系，空白处就是还没问到的地方。

| 账 | 我目前知道什么 | 还不知道什么 |
| --- | --- | --- |
| 身体 |  |  |
| 金钱 |  |  |
| 时间 |  |  |
| 退出 |  |  |

很多决定到这一步会自然慢下来。慢下来不是坏消息。身体不是闪购商品，少一点速度，常常是多一点自由。
