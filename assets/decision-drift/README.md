# 决策失重四源图组

这是一套原创的书内导航图。它参考 Cognitive Bias Codex 的一个问题意识：偏误并不是杂乱的名词，而是大脑在不同困难下采取的省力路径。

本图组没有复制原图的放射树、188 项目录、英文短语或节点结构。它将问题改写为医美消费中的四个决定时刻：

- 看见太多
- 解释太快
- 下手太急
- 回忆在改写

## 文件

- `svg/decision-drift-four-sources.svg`：总图，四源与 23 篇短章的入口。
- `svg/information-overload.svg`：看见太多。
- `svg/meaning-gap.svg`：解释太快。
- `svg/urgency-pressure.svg`：下手太急。
- `svg/memory-rewrite.svg`：回忆在改写。
- `raster/`：与 SVG 一一对应的 GPT Image 视觉伴读图。它们无文字，不承担图表信息。
- `spec.md`：唯一映射、文件名与视觉规格。
- `prompts.md`：五张栅格图的生成提示词与约束。
- `provenance.json`：栅格图生成与校验记录。
- `LICENSE.md`：原图来源与本项目图组的许可说明。
