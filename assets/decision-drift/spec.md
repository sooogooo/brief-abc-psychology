# 决策失重四源：唯一映射与视觉规格

## 这张图解决什么

它不是一张认知偏误的学术分类图，而是一本给医美消费者使用的导航图：当一个决定开始变轻、变快、变得难以退出时，先判断自己正落在哪一种困难里。

## 视觉规格

- 总图：1800 × 1400；分图：1600 × 1000。
- 总图使用 2 × 2 的四扇门卡片，中央为深色圆角矩形「决定失重」。
- 禁止圆心构图、放射线、枝状树、同心外环、原图英文标签及 188 项目录。
- 背景：`#F7F1E8`；正文：`#26313F`。
- 四源色：看见太多 `#5E8B7E`；解释太快 `#A97892`；下手太急 `#C9835A`；回忆在改写 `#9B9A66`。
- 字体：`Noto Sans CJK SC, Microsoft YaHei, sans-serif`；可读正文源字号不小于 32px。
- GitHub 约 900px 内容宽度下，主标题与分区名必须可读。约 375px 的移动端不依赖图片文字：每张图后均有等价的 Markdown 链接。

## 解释边界

四源是阅读路径，不是诊断，也不是严谨的学术分类。一个消费决定可以同时落入多个效应；图的用途只是帮读者找到先该停在哪一步。
