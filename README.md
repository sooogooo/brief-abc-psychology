# 简明使用医美消费心理学

> 不是劝人做，也不是劝人别做。它只解决一件事：在身体、钱和情绪都要付费的地方，把决定重新拿回自己手里。

医美消费最容易误判的，不是项目名，而是自己的状态。

有人明明想改善一处具体问题，却被引到一串项目里；有人在镜子前难受，进了咨询室就以为花钱能把难受一并解决；也有人效果并不差，却因为期待过高、恢复期难熬，急着追加项目。

这本短手册把已有的医美心理学、消费决策、营销伦理与咨询沟通材料收成一套 `ABC` 方法。每一章都能在一个具体时刻拿出来用。

## ABC 是什么

| 步骤 | 要做的事 | 最容易犯的错 |
| --- | --- | --- |
| A · Ask | 问清楚自己为什么现在想做 | 把别人的标准当成自己的需求 |
| B · Bias | 识别谁在推着自己下决定 | 把营销压力当成专业建议 |
| C · Check | 把身体、钱、时间和退出成本算清 | 只看价格和即时效果 |

## 这本手册不做什么

它不判断某个人该不该做哪一种治疗，也不替代面诊、知情同意或心理支持。项目是否合适，必须由具备相应资质的医疗专业人员结合个人情况判断。

它更像一把尺子：在决定之前，先量一量动机、信息、压力和自己能够承担的代价。
