# 焦虑营销识别表

看到一句话，先别急着相信它。看它在对什么下手。

| 话术或场面 | 它利用了什么 | 可以怎么回到事实 |
| --- | --- | --- |
| 「今天不定就没有这个价」 | 怕错过 | 「我明天再看，这个方案本身还成立吗？」 |
| 「你这里的问题很明显」 | 自我怀疑 | 「具体是什么问题，改善边界在哪里？」 |
| 「大家做完都很满意」 | 从众 | 「有没有不适合的人，或不满意时怎么处理？」 |
| 「这是对自己最值得的投资」 | 情感包装 | 「总成本、风险和退出成本分别是什么？」 |
| 「你现在不做，以后更麻烦」 | 年龄焦虑 | 「这属于医疗上的必要性，还是营销判断？」 |
| 「先做基础项目，后面再慢慢加」 | 追加消费惯性 | 「我现在只讨论一个需求，不打包决定后续项目。」 |

识别这些话术，不是为了把所有机构和咨询都当成陷阱。它只是把一件事还原：当信息不对称、情绪又很高时，消费者需要多一层自己的判断。

一句最够用的回应是：

> 我先把资料带走，想清楚再联系。

能说出这句话，很多不必要的决定就已经停住了。
